# YoutubeSyncPlayer

複数のYouTube動画を同時に再生し，タイミングを精密に同期させるWebアプリケーションです．
Google Antigravityを用いて作成されています．
このアプリケーションは，サーバーサイドの処理を必要としない静的Webサイトです．

##  機能

*   **同期調整**:
    *   ミリ秒単位でオフセットを調整します．
    *   **Tap Sync**: 音に合わせてクリックすることにより大まかな同期設定が可能です．Primary→Secondaryの順に押してください．
*   **自動ドリフト補正**:
    *   再生中の微妙なズレを検知し，自動的に同期を維持します．
    *   **Threshold（閾値）調整**: この数値以上のズレが検出された場合ドリフト補正が走ります．
*   **共有機能 (Share)**:
    *   動画の組み合わせとオフセット設定をURLパラメータとしてクリップボードにコピーします．

##  使い方

1.  **動画の追加**: YouTubeのURLを入力後，**Reload All**をクリックすることにより動画が表示されます．また，**Add Video**をクリックすると動画の枠が増えますが，3つ以上の同期再生については検証が不十分のため動作が不安定です．
2.  **PrimaryとSecondary**:
    *   最初の動画が **Primary** となります．
    *   2つ目以降は **Secondary** となり，Primaryに合わせて再生されます．
3.  **同期の調整**:
    *   Secondary動画の下にある**Sync**パラメーターによりオフセットを設定します．
4.  **Secondaryがカクつく場合**:
    *   画面下部の **Threshold** を調整してください．
    *   数値を大きくする（例: 100ms）と，許容が広がりカクつきが減りますが，同期精度は下がります．
    *   数値を小さくする（例: 40ms）と，厳密に同期しようとしますが，環境によっては頻繁にカクつきが発生します．
5.  **共有**:
    *   「Copy Link」ボタンを押すと，現在の状態を再現できるURLがクリップボードにコピーされます．

## 制限事項

*   モバイルOSには非対応です．これはOSの制限によるものであるためどうしようもありません．
